Sobre Minha Trajetória

Sou Allan Reymond, estudante de Ciência da Computação na Uninassau. Minha jornada na tecnologia começou no desenvolvimento de software, onde construí uma base sólida com Java, Python, JavaScript e frameworks como Angular e Node.js.

Durante esse período, dominei princípios fundamentais como SOLID, Clean Code e programação orientada a objetos. Hoje, aplico toda essa cultura de excelência técnica no universo de Dados.

Atualmente na Avanade, atuo em um modelo híbrido de Data Analyst e Data Engineer, focando em criar visualizações de dados impactantes com Power BI e tratar, processar e transformar dados utilizando Databricks, SQL, Python e Apache Spark.
