# 📊 Portfólio Profissional | Allan Reymond

Este repositório contém o código-fonte do meu portfólio pessoal, focado em **Data Engineering & Analytics**. O projeto foi desenvolvido com uma estética moderna, premium e altamente responsiva, destacando minhas competências técnicas e trajetória profissional.

## 🚀 Tecnologias Utilizadas

- **Angular 18+** (Standalone Components & Signals)
- **Tailwind CSS** (Design System e Estilização)
- **Lucide Angular** (Ícones Vetoriais)
- **TypeScript** (Lógica e Tipagem)
- **HTML5/CSS3** (Estrutura Semântica)

## ✨ Funcionalidades

- **Dark Mode Support**: Design otimizado para visualização em ambientes escuros.
- **Micro-interações**: Feedback visual em cards, botões e links.
- **Seções Detalhadas**:
  - **Hero**: Destaque para certificações Microsoft Azure.
  - **Sobre**: Trajetória do desenvolvimento de software ao mundo dos dados.
  - **Stack Técnica**: Filtro visual das tecnologias dominadas (Power BI, SQL, Python, Databricks, etc).
  - **Experiência**: Linha do tempo profissional e treinamentos.
  - **Educação e Certificações**: Listagem organizada de formação acadêmica e badges Azure.
  - **Contato Direto**: Canais simplificados via cards interativos.

## 🛠️ Como Executar o Projeto Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/AllanRDS/Portfolio.git
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   # ou
   ng serve
   ```

4. **Acesse no navegador:**
   `http://localhost:4200`

## 📂 Estrutura do Projeto

- `/src/app/landing-page`: Contém o componente principal, template HTML e estilos específicos.
- `/public/assets`: Armazena imagens de perfil, currículo PDF e ícones.
- `/src/styles.css`: Configurações globais e importação do Tailwind.
